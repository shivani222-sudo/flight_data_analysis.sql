# flight_data_analysis.sql
SQL-based flight data analysis covering passenger trends, route analysis, and city-level air traffic insight

## Project Overview
This project analyzes airline and airport data to identify high-traffic routes, passenger trends, and city-level air traffic patterns using SQL. The objective is to generate actionable insights to support capacity planning and operational decision-making.

## Dataset
The dataset contains flight-level information including airlines, origin and destination airports, passenger counts, freight, and mail volumes. Data was cleaned and normalized before analysis.

## Tools & Technologies
- SQL (MySQL)
- Relational Database Design
- Data Cleaning & Aggregation

## Key Business Questions Answered
- Which routes and corridors have the highest passenger traffic?
- How does passenger volume vary by month and year?
- Which cities and airports handle the most air traffic?
- Is there a relationship between city population and air traffic volume?
